## Install
Try it out in your browser [here]().

Or to install:
``` bash
% git clone https://github.com/arthurfincham/birthday_app
```

## Usage

Go to the project directory and install dependencies:

``` ruby
% bundle install
 ```

Run the app with Sinatra"
``` ruby
% ruby app.rb
```
and you're off!
